Dashboard Guide

What is this?
The AI Pulse Dashboard is a centralized status monitor for various AI API providers. It tracks availability, supported features, and model lists for major providers like OpenAI, Anthropic, Google Gemini, and many others.
How to Test an API
  • Step 1: Find a provider in the list. Use headers to sort.
  • Step 2: Click "Test" on the right.
  • Step 3: Select the Model you want to query.
  • Step 4: Enter your own API Key. (Processed locally, never saved).
  • Step 5: Click "Run Test".
