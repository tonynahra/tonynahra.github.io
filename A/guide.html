MediaMaze Viewer

A High-Performance Digital Album Experience

The MediaMaze Viewer is not just a gallery; it is a presentation engine designed to turn folders of images into immersive, cinematic experiences. It prioritizes speed, keyboard efficiency, and audio-visual flow over traditional point-and-click interfaces.

Powered by the Album Builder

This viewer works in tandem with the MediaMaze Album Builder. The Builder is an intelligent backend tool that scans your folders, uses AI to generate descriptions and tags, and compiles everything into the highly optimized JSON format this viewer consumes.

(Contact your administrator to access the Album Builder)

For the best experience, use your keyboard. The interface is designed to fade away, leaving only your content.

The viewer supports a unique "Annotated Notes" layer. This is perfect for technical diagrams, "before/after" shots, or detailed text overlays.

How it Works

If a photo has associated notes, a üìù icon appears in the HUD.

  • Press N to toggle.
  • It cycles through: Original Image ‚Üí Note 1 ‚Üí Note 2 ‚Üí ... ‚Üí Original.
  • A persistent indicator (e.g., "Note 1 / 3") will appear in the top-right corner to show your position in the stack.

Note: Navigating to a new photo automatically resets the view to the "Original" image.

Slideshows

Press any number key from 1 to 9 to start an instant slideshow.

  • 1 : 3 Seconds
  • 5 : 7 Seconds (Standard)
  • 9 : 11 Seconds
  • 0 / Esc : Stop

Transition Effects

The viewer includes a cinema-grade transition engine (Fade, Zoom, Slide, Flip, Bounce, etc.).

  • Random (Default): The viewer picks a new effect for every photo.
  • Fade Only: Press E to enforce a strict "Fade" transition for a calmer experience. Press again to return to Random.

You can create "Smart Links" that launch the album in a specific state by adding a hash (#) followed by options.

Format: https://site.com/viewer/?AlbumName#Option1,Option2...

Available Options

  • #S : Silent Mode (No popups, auto-hide cursor).
  • #N : Start with Notes enabled.
  • #R : Randomize the photo order on load.
  • #F : Fullscreen (Enables on first click).
  • #E[0-9] : Enforce Effect (e.g., #E1 for Fade).
  • #[Digit] : Auto-Start Slideshow (e.g., #5).
  • #CAT-[Name] : Filter by Category.
  • #KW-[Word] : Filter by Title Keyword.
  • #ID : Start at specific Photo ID.
