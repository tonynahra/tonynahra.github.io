MediaMaze Viewer Guide

The definitive guide to creating and viewing dynamic albums.

The MediaMaze Viewer is a lightweight, high-performance web application designed to turn simple folders of images into immersive, interactive storytelling experiences.

Unlike traditional galleries, it focuses on keyboard navigation, background audio, and fluid animations to create a presentation-like feel.

For Creators: You don't need to write code. The viewer reads a simple JSON file (generated by our AI tools) to understand your album structure, music, and notes.

The viewer is optimized for keyboard use, making it perfect for presentations or lean-back viewing.

  • Next/Prev: Use Right / Left arrows or Space.
  • Info Panel: Use Up / Down arrows to cycle through view modes: Hidden ‚Üí Bottom Bar ‚Üí Full Details Modal.
  • Categories: Use PageUp / PageDown to cycle through photo categories instantly.
  • Reset: Press Home to clear all filters and return to the first photo.
  • Slideshow: Press 1 through 9 to start an auto-play (3s to 11s delays). Press 0 or Esc to stop.

You can customize how an album starts by adding special "Hash Options" to the end of the URL. Multiple options can be comma-separated.

Example: .../AlbumName#R,S,5 (Randomize, Silent Mode, 8s Slideshow)

  • #S : Silent Mode. Disables toast notifications and hides mouse cursor in fullscreen.
  • #N : Notes Mode. Starts with "Annotated Notes" view enabled by default.
  • #R : Randomize. Shuffles the photo order on load.
  • #F : Fullscreen. Enters fullscreen on the first click.
  • #[Digit] : Auto-Start. Starts slideshow. #1 = 3s, #9 = 11s.
  • #CAT-[Name] : Filter Category. e.g., #CAT-Nature.
  • #KW-[Word] : Filter Keyword. e.g., #KW-Mountain filters by title.
Pro Tip: Press A inside the viewer to open the Link Assistant. It lets you select these options visually and generates the link for you!

Advanced users can create their own albums by generating a JSON file in the same directory as their photos.

Required Structure:

{
  "albumTitle": "My Trip",
  "photos": [
    {
      "url": "image1.jpg",
      "title": "The Beach",
      "categories": "Nature, Summer",
      "notes": ["image1_note.png"]
    }
  ]
}
